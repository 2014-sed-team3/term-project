Keyboard Shortcuts in the Graph Pane

Several keyboard shortcuts are available in NodeXL's Graph Pane. Most of them are alternatives to using the Ribbon or the Graph Pane's right-click menu.

Important Note: You must click the Graph Pane before you can use any of these keyboard shortcuts.

To Do This Press This
Show the graph Ctrl+R
Lay out the entire graph without rereading the workbook Ctrl+L
Select all vertices Ctrl+V
Select all edges Ctrl+E
Select all vertices and edges Ctrl+A
Deselect all vertices and edges Ctrl+D
Toggle the selection Ctrl+T
Edit the properties of the selected vertices Ctrl+P
Copy an image of the graph to the Clipboard Ctrl+C
Save an image of the graph to a file Ctrl+I
Move the selected vertices a small distance Left Arrow
Right Arrow
Up Arrow
Down Arrow
Move the selected vertices a large distance Shift+Left Arrow
Shift+Right Arrow
Shift+Up Arrow
Shift+Down Arrow
