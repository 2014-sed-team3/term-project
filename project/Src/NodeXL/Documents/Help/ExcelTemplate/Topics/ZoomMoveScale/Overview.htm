Zooming, Moving Around and Scaling the Graph

NodeXL's Graph Pane usually shows you the entire graph. If the graph is simple, viewing it in its entirety may be exactly what you want. But if the graph is more complex, you might want to zoom in to examine specific regions more closely.

NodeXL lets you zoom into and move around the graph using several techniques. You can also scale the graph, which shrinks all the graph's elements while keeping them in the same place.
