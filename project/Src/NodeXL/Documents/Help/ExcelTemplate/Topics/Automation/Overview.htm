Automating Common Tasks

If you find yourself repeatedly running the same sequence of tasks on a NodeXL workbook, you might find NodeXL's automation feature useful. With automation, you can run any or all of the following tasks with a single button click:

  • Merge duplicate edges
  • Create groups by cluster
  • Calculate graph metrics
  • Autofill columns
  • Create subgraph images
  • Show the graph
  • Save an image of the graph

You can run the tasks on either the current workbook or on every NodeXL workbook in a folder that you specify.

To automate common tasks:
  1. In the Excel Ribbon, select NodeXL, Graph, Automate.
  2. In the Automate dialog box, check the tasks you want to automate.
To change the options for an automation task:
  1. In the Automate dialog box, select the button to the right of the task ().
To automatically automate tasks after data is imported into the workbook:
  1. In the Excel Ribbon, select NodeXL, Data, Import, Import Options.
  2. In the Import Data Options dialog box, check the "automate the graph after the data is imported" checkbox.
  3. In the Import Data Options dialog box, select the "automate options" button () to select the tasks you want to automate after data is imported.
