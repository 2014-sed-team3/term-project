Layout Options

You can change the way NodeXL lays out the graph with a number of layout options:

  • If your graph has groups, you can tell NodeXL to lay out each group in its own box.
  • If your graph has a large number of small connected components that obscure the larger, more important components, you can tell NodeXL to move the smaller components to the bottom of the Graph Pane.
  • If you have selected the Fruchterman-Reingold layout algorithm, you can control how that algorithm works.
To set layout options, do one of the following:
  • In the Excel Ribbon, select NodeXL, Graph, Layout, Layout Options, or...
  • At the top of the Graph Pane, select Layout Options from the Layout drop-down list ().
