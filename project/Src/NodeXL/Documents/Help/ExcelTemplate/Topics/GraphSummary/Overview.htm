Summarizing the Graph

If you create a large number of graphs, NodeXL's graph summary feature might be useful to you.

NodeXL automatically keeps track of several aspects of how a graph was created—how it was laid out, how its visual properties were set using Autofill Columns, how its vertices were grouped, and so on. It will provide you with a summary of this information, which you can use for your own documentation.

To get a summary of the graph:
  • Go to NodeXL, Graph, Summary in the Ribbon.

Note that for privacy reasons, NodeXL does not keep track of the most recent data that was imported into the workbook unless you tell it to.

To tell NodeXL to keep track of the data that gets imported into the workbook:
  1. Go to NodeXL, Data, Import in the Ribbon.
  2. Check the Save Import Details in Graph Summary menu item.
To apply this change to all new workbooks:
  1. In the Excel Ribbon, select NodeXL, Options, Use Current for New.
  2. When you are asked for confirmation, answer "yes."
