A Quick Tour of the NodeXL Workbook

When you open the NodeXL Excel Template from the Windows Start Menu, Excel creates a new workbook from the template and opens the workbook. This NodeXL workbook differs from a standard Excel workbook in that it contains multiple worksheets that have been customized for network graph data, a custom Graph Pane used to display the network graph, and a custom NodeXL tab in the Excel Ribbon that contains commands specific to NodeXL.

These features appear only in NodeXL workbooks, and they are removed entirely from Excel when you close the NodeXL workbook.1

  1. It's the transient nature of these features that makes the NodeXL Excel Template (and any Excel template, in fact) different from an Excel add-in, which is a set of features that are available at all times, from any Excel workbook. NodeXL is not an Excel add-in.
