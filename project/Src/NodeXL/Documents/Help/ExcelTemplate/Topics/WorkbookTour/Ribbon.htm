The NodeXL Ribbon Tab

The NodeXL workbook adds a custom NodeXL tab to the Excel Ribbon. This tab contains all the commands that are specific to NodeXL graphs, organized into groups. It does not appear in standard Excel workbooks.

All the standard Excel Ribbon tabs (Home, Insert, Page Layout, and so on) are available in NodeXL workbooks as well.

To learn more about a command in the NodeXL Ribbon tab:
  1. Hover the mouse over the command.
