Creating Groups

You can tell NodeXL to automatically create a set of groups using one of several techniques. You can also create groups by manually selecting vertices and giving them a group name.

In either case, creating groups inserts the new groups into the workbook (see Understanding the Group Worksheets). The groups do not automatically appear in the Graph Pane, however. To see the groups in the Graph Pane, you must show the graph after creating your groups.

See How Groups are Shown in the Graph Pane for information on how NodeXL shows groups.
