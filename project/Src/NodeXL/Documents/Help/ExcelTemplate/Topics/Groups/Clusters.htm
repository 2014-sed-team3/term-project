Creating Groups by Cluster

A cluster is a group of vertices that are densely connected to one another but sparsely connected to the rest of the graph. NodeXL uses one of several mathematical algorithms to analyze the connection density and create a set of groups accordingly.

To remove any existing groups, create a group for each of the graph's clusters, and put every vertex in one of the new groups:
  1. In the Excel Ribbon, select NodeXL, Analysis, Groups, Group by Cluster.
  2. In the Group by Cluster dialog box, select a cluster algorithm to use. The speed and effectiveness of the algorithms vary with the particular graph. You may want to experiment with the algorithms to determine which you prefer.
  3. Specify what you want to do with vertices that have no neighbors.
  4. Show the graph to see the groups.
