Labeling Edges, Vertices and Groups

You can add edge and vertex labels to the graph. An edge label appears on top of the edge, while a vertex label can appear either inside a Label shape that represents the vertex or as an annotation next to the vertex.

If your graph has groups, you can label the groups. Group labels appear in the graph when you choose to lay out the groups in boxes, and when you collapse groups.

To label edges:
  1. Select the Edges worksheet.
  2. If the worksheet's Visual Properties column group is not visible, show the column group.
  3. For each edge that should have a label, enter the label text in the Label column.
  4. Show the graph to make your changes appear in the Graph Pane.
To label vertices using the Label shape:
  1. Select the Vertices worksheet.
  2. If the worksheet's Visual Properties column group is not visible, show the column group.
  3. For each vertex that should have a Label shape, enter Label in the Shape column and enter the label text in the Label column.

  1. Select the Vertices worksheet.
  2. If the worksheet's Visual Properties column group is not visible, show the column group.
  3. For each vertex that should be annotated with a label, enter the label text in the Label column.

To label groups:
  1. Select the Groups worksheet.
  2. If the worksheet's Visual Properties column group is not visible, show the column group.
  3. For each group that should have a label, enter the label text in the Label column.
  4. Show the graph to make your changes appear in the Graph Pane.
