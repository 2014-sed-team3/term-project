Selecting Graph Elements

You can select one or more of the graph's vertices and edges. Selected vertices and edges are highlighted in the NodeXL worksheets and in the Graph Pane. You can change the color of the selected vertices and edges in the Graph Pane using Graph Options.

If your graph has groups, you can select an entire group at once. See Selecting Groups for details.
